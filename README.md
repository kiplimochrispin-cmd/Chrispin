# Eldoret National Polytechnic Website (TypeScript + GSAP)

Responsive website project for **Eldoret National Polytechnic (TENP)** built with Vite, React, TypeScript, and GSAP.

## Pages
- Home (`/`)
- Location (`/location`)
- Contact Us (`/contact`)

## Features
- Mobile-first responsive navigation and layout
- GSAP animations for hero and content reveals
- School overview content based on official TENP information
- Location details + embedded map view
- Contact page with official phone/email and inquiry form UI

## Run locally
```bash
cd tenp-website
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
